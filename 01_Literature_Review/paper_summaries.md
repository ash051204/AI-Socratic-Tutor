# Literature Review Summary

## 1. Automated Socratic Tutoring via Question Generation (2021)

### Objective
The study focuses on generating Socratic-style questions using automated techniques to support structured reasoning in students.

### Approach
The system primarily relies on rule-based templates to generate layered questions that guide learners step-by-step.

### Strengths
- Clearly structured questioning strategy
- Emphasis on scaffolding reasoning
- Strong theoretical alignment with Socratic pedagogy

### Limitations
- Limited adaptability
- Not based on modern transformer models
- Rigid template-based generation

### Relevance
The paper demonstrates the value of structured questioning but highlights the need for more adaptive, LLM-driven systems.


## 2. Conversational AI Tutors for Inquiry-Based Learning (2023)

### Objective
To explore how conversational agents can promote inquiry-based learning through interactive dialogue.

### Approach
The system uses dialogue-driven prompts to encourage reflection and reasoning.

### Strengths
- Encourages student participation
- Improves engagement compared to static tutoring systems

### Limitations
- Limited personalization
- Conceptual depth varies across subjects

### Relevance
Supports the idea that conversational agents can enhance learning but shows the need for structured adaptation.


## 3. Applying Bloom’s Taxonomy for Adaptive Questioning (2020)

### Objective
To design adaptive questioning models based on hierarchical learning levels.

### Approach
Questions are categorized according to Bloom’s cognitive levels to promote progressive learning.

### Strengths
- Clear difficulty progression
- Supports structured learning depth

### Limitations
- Mostly theoretical
- Does not integrate AI-based conversational systems

### Relevance
Provides a strong framework for designing adaptive difficulty logic in the proposed system.
