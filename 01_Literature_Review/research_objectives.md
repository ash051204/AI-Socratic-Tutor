## Research Objectives

The main objectives of this project are as follows:

- To design a conversational AI teaching assistant that follows the Socratic Method by engaging students through structured, question-driven dialogue instead of direct answers.

- To develop a question generation module based on Bloom's Taxonomy that can generate questions across six levels: recall, comprehension, application, analysis, evaluation, and creation, and select the appropriate level based on student performance.

- To implement a learner response evaluation system that assesses student answers using semantic similarity and reasoning quality, rather than relying on fixed answer keys.

- To build an adaptive difficulty controller that tracks student performance and dynamically adjusts the difficulty level, increasing complexity when the student performs well and simplifying when needed.

- To integrate all components into a unified conversational system using a large language model, ensuring smooth interaction, context awareness, and meaningful follow-up questions.

- To evaluate the performance of the system across different educational topics by analyzing both the quality of generated questions and their effectiveness in improving student understanding.
