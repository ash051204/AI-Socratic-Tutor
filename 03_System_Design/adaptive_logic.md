# Adaptive Difficulty Logic

The system adapts its questioning depth based on student responses.

- If the student struggles, the system shifts to simpler cognitive levels.
- If the student responds confidently, the system progresses toward deeper analytical levels.
- If misconceptions are detected, the system provides guiding hints instead of corrections.

Bloom’s Taxonomy levels used:
Remember → Understand → Apply → Analyze → Evaluate → Create

This adaptive mechanism ensures that learning remains personalized and structured.
