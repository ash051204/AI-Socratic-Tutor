# Proposed System Architecture

The proposed AI teaching assistant consists of the following conceptual components:

## 1. Input Understanding Module
Accepts student queries and analyzes language patterns to estimate conceptual familiarity and difficulty level.

## 2. Socratic Question Generator
Produces layered and structured questions instead of direct explanations. Questions may include clarification prompts, counterexamples, or application-based scenarios.

## 3. Adaptive Reasoning Engine
Tracks student responses and adjusts difficulty according to Bloom’s Taxonomy levels.

## 4. Dialogue Manager
Maintains conversational flow and ensures that the system avoids revealing answers prematurely.

## 5. Reflection Module
Concludes interaction with a reasoning-based summary to reinforce learning outcomes.
