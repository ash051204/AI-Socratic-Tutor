# Socratic Question Framework

The questioning structure follows a progressive reasoning pattern:

1. Clarification Questions  
   Example: "What do you already know about this concept?"

2. Why-Based Questions  
   Example: "Why do you think this occurs?"

3. Counterexample Prompts  
   Example: "What happens if we change one of the conditions?"

4. Application Questions  
   Example: "Where might this concept be used in practice?"

5. Reflective Questions  
   Example: "How would you explain this idea in your own words?"

This structure encourages active reasoning rather than passive answer reception.
