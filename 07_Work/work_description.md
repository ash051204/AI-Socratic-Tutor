##  Work Description

The practical work for this project involved the design, development, and testing of a prototype AI Teaching Assistant based on the Socratic Method. The work was carried out in four main phases over the semester.

---

### Phase 1: Literature Survey

An extensive literature review was conducted covering ten research papers related to:

- Automatic Question Generation (AQG)  
- Intelligent Tutoring Systems (ITS)  
- Large Language Models (LLMs)  

This phase included:

- Reading and summarizing research papers  
- Identifying key techniques, datasets, and evaluation methods  
- Performing a comparative analysis  

The insights gained from this phase helped in identifying the research gap and guiding the system design.

---

### Phase 2: System Design

In this phase, the overall system architecture was designed.

Key decisions included:

- Selection of LLM (GPT-4 used in prototype)  
- Implementation of Bloom’s Taxonomy for question levels  
- Use of sentence-transformers for semantic similarity  
- Design of adaptive difficulty algorithm  

The system design was documented using diagrams and pseudocode.
