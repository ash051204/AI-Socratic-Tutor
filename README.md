# AI Teaching Assistant Using the Socratic Method

## Project Overview
This project explores the design of an AI-based teaching assistant that supports conceptual learning through structured questioning rather than direct answer delivery. The aim is to encourage students to think critically, reflect on their reasoning, and build deeper understanding of concepts.

## Background
Many existing AI learning tools provide instant answers to student queries. While this approach is convenient, it often leads to surface-level learning and limited conceptual clarity. Students may memorize solutions without fully understanding the reasoning behind them.

The Socratic method, a classical teaching approach based on guided questioning, promotes active reasoning and reflection. This project studies how such a method can be integrated with modern Large Language Models (LLMs) to create an adaptive, inquiry-based AI tutor.

## Problem Statement
There is a growing dependence on direct-answer AI systems in education, which can weaken analytical thinking and conceptual understanding. A structured, adaptive questioning framework is required to promote deeper learning.

## Proposed Approach
The proposed system combines:
- Socratic questioning techniques
- Bloom’s Taxonomy for structured learning progression
- LLM-based conversational capability
- Adaptive reasoning based on student responses

The system is designed to guide learners step-by-step instead of revealing solutions immediately.
